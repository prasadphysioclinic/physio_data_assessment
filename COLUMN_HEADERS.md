# Google Sheets Column Headers

Copy and paste this line into Row 1 of your Google Sheet (A1 to AL1):

```
Date	PatientName	Age	Occupation	MechanismOfInjury	AggravatingEasingFactors	TwentyFourHourHistory	ImprovingStaticWorse	NewOrOldInjury	PastHistory	DiagnosticImaging	PainLocation	BodyMapImage	PainIntensity_VAS	PainPattern	ObservationPosture	Active_L_Flex	Active_R_Flex	Active_L_Ext	Active_R_Ext	Passive_L_Flex	Passive_R_Flex	Passive_L_Ext	Passive_R_Ext	EndFeel	CapsularPattern	ResistedIsometrics	FunctionalTesting	SensoryScan	Reflexes	NeuroSpecialTests	SpecialTests	JointPlayMovements	Palpation_Tenderness	Palpation_Effusion	Comments	SubmittedBy	Timestamp
```

## Column Descriptions

| Column | Description |
|--------|-------------|
| Date | Assessment date |
| PatientName | Patient's full name |
| Age | Patient's age |
| Occupation | Patient's occupation |
| MechanismOfInjury | How the injury occurred |
| AggravatingEasingFactors | What makes it better/worse |
| TwentyFourHourHistory | 24-hour symptom pattern |
| ImprovingStaticWorse | Current status (Improving/Static/Worse) |
| NewOrOldInjury | New or Old injury |
| PastHistory | Previous medical history |
| DiagnosticImaging | X-ray, MRI, CT scan results |
| PainLocation | Where the pain is located |
| BodyMapImage | Body chart image URL (future feature) |
| PainIntensity_VAS | Pain scale 0-10 |
| PainPattern | Constant, periodic, episodic, occasional |
| ObservationPosture | Posture observations |
| Active_L_Flex | Left active flexion |
| Active_R_Flex | Right active flexion |
| Active_L_Ext | Left active extension |
| Active_R_Ext | Right active extension |
| Passive_L_Flex | Left passive flexion |
| Passive_R_Flex | Right passive flexion |
| Passive_L_Ext | Left passive extension |
| Passive_R_Ext | Right passive extension |
| EndFeel | End feel assessment |
| CapsularPattern | Capsular pattern present |
| ResistedIsometrics | Resisted isometric test results |
| FunctionalTesting | Functional test results |
| SensoryScan | Sensory examination results |
| Reflexes | Reflex test results |
| NeuroSpecialTests | Neurological special tests |
| SpecialTests | Other special tests |
| JointPlayMovements | Joint play assessment |
| Palpation_Tenderness | Tenderness on palpation |
| Palpation_Effusion | Effusion/swelling findings |
| Comments | Additional comments |
| SubmittedBy | Who submitted the assessment |
| Timestamp | When the assessment was submitted |
